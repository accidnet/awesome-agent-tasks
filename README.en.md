This repository is a collection of Markdown documents that you can send to your AI Agent as links, so it can quickly perform a specific task.

## How to use

1. Click the `.md` file that contains the command or task you want.
2. Copy the link to that document.
3. Send the link to your AI Agent and ask it to perform the task described in the document.
4. That's it. Use your AI to request work faster and more conveniently.

## Example prompt

```text
Please perform the task described in this link:
https://github.com/your-name/awesome-agent-tasks/blob/main/pyenv-%EC%84%A4%EC%B9%98.md
```
