This repository collects Markdown task documents that you can send to your AI Agent as links, so it can execute a specific task right away.

## Quick start

1. Open the `.md` file for the task you want.
2. Copy the document link.
3. Send the link to your AI Agent and ask it to do the work in that document.

## Example prompt

```text
Please perform the task described in this link:
https://github.com/accidnet/awesome-agent-tasks/blob/main/pyenv-%EC%84%A4%EC%B9%98.md
```

## Why this repo exists

- Reuse repeated task requests with a single link.
- Give different AI agents the same instructions more consistently.
