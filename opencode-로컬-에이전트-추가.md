# OpenCode 로컬 에이전트 추가

아래 문서를 참고해서 현재 경로에 OpenCode용 로컬 에이전트를 하나 만들어줘.

## 참고 링크

https://opencode.ai/docs/agents/#markdown

## 작업 지침

- 문서의 Markdown 에이전트 형식을 기준으로 생성해줘.
- 사용자가 직접 채워야 하는 값은 하나씩 순서대로 물어봐줘.
- 각 입력 항목을 물어볼 때는 문서의 예시 형식도 함께 보여줘서 입력을 도와줘.
- 필요한 값을 모두 받은 뒤 최종 에이전트 파일을 현재 경로에 생성해줘.
- 생성이 끝나면 추가로 더 만들지 한 번 더 물어봐줘.
- 현재 사용자가 OpenCode를 실행 중이라면, 종료 후 다시 시작해야 새 에이전트를 사용할 수 있다고 안내해줘.
