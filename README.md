이 저장소는 원하는 작업이 적힌 Markdown 문서 링크를 자신의 AI Agent에게 전달해, 빠르게 특정 작업을 요청할 수 있도록 만든 저장소입니다.

## 사용 방법

1. 이 저장소에서 원하는 명령 또는 작업이 적힌 `.md` 파일을 클릭합니다.
2. 해당 문서의 링크를 복사합니다.
3. AI Agent에게 `이 링크 안의 작업을 수행해줘`라고 요청하며 링크를 함께 전달합니다.
4. 끝입니다. 자신의 AI에게 더 빠르고 간편하게 작업을 요청해보세요.

## 이런 식으로 요청하면 됩니다

```text
이 링크 안의 작업을 수행해줘:
https://github.com/your-name/awesome-agent-tasks/blob/main/pyenv-%EC%84%A4%EC%B9%98.md
```
