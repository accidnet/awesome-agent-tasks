링크만 전달하면 AI Agent가 바로 작업을 수행할 수 있도록,
작업 지시를 Markdown 문서로 모아둔 저장소입니다.

## 빠른 시작

1. 원하는 작업이 적힌 `.md` 파일을 엽니다.
2. 해당 문서 링크를 복사합니다.
3. AI Agent에게 링크와 함께 작업을 요청합니다.

## 예시 프롬프트

```text
이 링크에 적힌 작업을 수행해줘:
https://github.com/accidnet/awesome-agent-tasks/blob/main/pyenv-%EC%84%A4%EC%B9%98.md
```

## 목적

- 반복적으로 요청하는 작업을 링크 하나로 재사용할 수 있습니다.
- Agent마다 같은 작업을 더 일관된 방식으로 전달할 수 있습니다.
